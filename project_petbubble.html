Game Project

Pet the Bubble

A game jam game about petting a soap bubble!

This game was made as part of Global Game Jam 2025, over a single weekend. As I was quite busy that particular weekend, I decided to keep the scope super minimal and create a game where you simply pet a bubble.

The theme of the gamejam was "bubbles", and the idea came to mind of creating a game about trying to pet something that is inherently fragile.

This was the first game I ever completed, exported and published, and it taught me a lot about priotitizing the features that are absolutely necessary to create a playable experience.

The game can be played in browser here at itch.io!

Gameplay

The game does not give much info to the player regarding the mechanics, aside from how to pet the bubble. It is up to the player to experiment in order to learn how to win and lose the game - or whether such a thing is even possible.

The animations and bubble animations react to being pet, as an internal score keeps track of the bubbles happiness level.

There are two fail states to the game:

  • Petting the bubble too much
  • Petting the bubble too rarely

A cooldown timer keeps track of the bubbles "stress" level and "loneliness" level, and if either value reaches a certain point it is game over.

Hoever, if the player manages to pet the bubble just the right amount, they are rewarded with a victory screen and a nice song.


The face follows the mouse cursor as the bubble waits in anticipation! Meanwhile, you can pop nearby (non-sentient) bubbles.

A lot of work went into making sure the face and bubble animations felt responsive and reactive. The face is also the primary communication tools regarding the state of the bubble. It is up to the player to recognize the changing face and adjust their petting tempo accordingly.
