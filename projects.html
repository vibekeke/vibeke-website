Pet the Bubble Game

Godot Engine gdscript Solo Project Global Game Jam

A short game jam for the Global GameJam 2025, with the theme "bubbles"

The game was made entirely by me over a single weekend. My goal for this project was to keep things extra simple so that I could finish it despite the limited timeframe.

I made a lot of mistakes on this project and learned a lot, and I am quite satisfied with the fact that I was able to finish the game! The game can be played in browser, and although it works on mobile it is intended to be played on desktop.

The Colors of Grandma's forest

Python PyGame Team Project

A short RPG made for the IFI-Gamejam 2025. The game was made over a single weekend, in a team of 6 students. My main contribution to the game was the scene switching logic, as well as general integration of different game parts.

The topic of the Game Jam was "Fairytale." We decided to make a story based adventure about the young girl, Iris, who helps her sweet grandma find her lost belongings. But, as so often happens in fairytales, doom looms over the land. Colors are disappearing from the world, and Grandma is behaving suspiciously. Iris must embark on a journey across the forest and Grandma's garden to save both the world and her beloved granny.

The game is made entirely in PyGame, and the visuals were made in libresprite. The game requires Python and the pyGame library to play.
